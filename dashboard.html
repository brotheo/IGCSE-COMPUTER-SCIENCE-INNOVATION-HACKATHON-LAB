Program Objectives

By the conclusion of this program, students will demonstrate competency in the following areas:

Academic Mastery

Understand fundamental IGCSE Computer Science programming concepts.

Computational Thinking

Apply algorithm design and logical problem-solving strategies.

Technical Proficiency

Write structured code using variables, conditions, loops, and functions.

Quality Assurance

Debug, test, and refine programs effectively.

Collaboration

Work collaboratively within a small development team.

Hackathon Innovation

Design, build, and present a functional web application in a team-based hackathon.

Communication

Present technical projects confidently in a hackathon setting.

Team Roles (Rotate Every 3-4 Weeks)

Team Lead

Organizes tasks, keeps team on track generally, and coordinates the entire workflow.

Programmer

Writes code, implements core features, and develops programming logic.

Resources & Guidelines

Technical Requirements

Computers with internet (1 per student), VS Code Editor, Modern web browser, Projector.

Instructor Tips

Encourage peer learning. Allow error-based learning during debugging. Celebrate small wins. Balance theory with practice.

Differentiation

Provide extension activities for advanced students, and support for struggling learners. Use peer mentoring.

Note to Parents

Skill Development Over Competition

This program prioritizes learning and skill acquisition above competition rankings. The hackathon component is designed to provide students with experience in applying their knowledge, rather than pressuring them to create highly complex commercial software.

Long-Term Value

Students will develop problem-solving, logical thinking, creativity, and teamwork. These are transferable skills essential for Computer Science examinations and valuable across many future career paths.
